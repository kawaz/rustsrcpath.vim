# vim-rustsrcpath

Get rust's ssource, and set `$RUST_SRC_PATH`.

## Install

```vim:~/.vimrc
call dein#add('kawaz/vim-rustsrcpath', {'on_ft': ['rust']})
" Add syntastic sh_sh_checker for *.bats
```
